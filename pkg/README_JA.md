# pkgディレクトリ

📖 **English version**: [README.md](./README.md)

このディレクトリには、外部アプリケーションから安全にインポートできるライブラリコードが含まれています。ここに配置されるコードは、よく設計され、安定しており、公開使用を意図したものである必要があります。

**ガイドライン:**
- 他のプロジェクトがインポートできる再利用可能なパッケージを配置する
- APIが十分に文書化され、安定していることを確認する
- 内部実装の詳細を避ける
- Goパッケージの命名規則に従う

Goプロジェクトのレイアウトとパッケージ構成の詳細については、[公式Goドキュメント](https://golang.org/doc/code.html)と[プロジェクトレイアウト標準](https://github.com/golang-standards/project-layout#pkg)を参照してください。
